My Projects

Car Detection with YOLO

Built a real-time object detection system using the YOLO deep learning model to accurately identify and localize cars in various image conditions. The model was trained and optimized on a custom car dataset, incorporating techniques like IoU analysis and anchor box tuning for precise bounding box predictions.

Trigger Word Detection

Developed a wake-word detection system inspired by voice assistants like Siri and Alexa, using LSTM-based neural networks to identify the trigger word “activate” in audio recordings. Created a custom speech dataset and integrated real-time audio input with a chime-based feedback system to simulate smart assistant responsiveness.

Transfer Learning with MobileNet

Utilized transfer learning with the MobileNet architecture to develop a lightweight image classifier that distinguishes between Alpaca and non-Alpaca images. This project demonstrated the power of pre-trained convolutional networks in achieving high accuracy with minimal training data.

AI Tic Tac Toe

Created a web-based Tic Tac Toe game with an unbeatable AI opponent using the Minimax algorithm and alpha-beta pruning for optimal move calculation. The game features an interactive Streamlit interface with real-time gameplay and sub-50ms AI responses for a seamless user experience.

Web Automation and Scraping Tool

Engineered an end-to-end automation bot that scrapes data from websites and uploads it to spreadsheets using Sheety API. Integrated Python, Selenium, BeautifulSoup, and a Tkinter-based GUI to streamline the data collection and processing pipeline.
